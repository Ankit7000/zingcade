How it plays

Merge Monster 2048 keeps the proven slide-and-merge structure but shifts the focus to creature evolution. Every valid move spawns a new basic tile, every merge climbs the ladder, and every dead move tightens the board.

Controls

  • Desktop: Arrow keys are always active, and WASD also works.
  • Mobile: Swipe inside the board. Touch scroll is blocked while you play.
  • Restart: Tap the restart button or press R for an instant reset.

Evolution ladder

  1. Egg to Slime starts every run.
  2. Goblin, Wolf, and Golem stabilize the mid-board.
  3. Mage and Dragon are the serious climb gates.
  4. Ancient Dragon, Mythic Beast, and Void Titan separate clean boards from lucky ones.
Rank preview support

Bronze through Legend is calculated locally for now from best score and best evolution. The UI is structured so a shared Zingcade profile system can replace the local profile later.

Daily challenge support

Daily targets already rotate and store completion locally. Seeded starting boards can plug into the same challenge object later without changing the board engine.

What It Is

Creature-first merge puzzle.

The board reads like a clean arcade puzzle cabinet, not a plain number grid. Names, marks, and tier color carry the run.

Replay Hook

Short runs, stubborn mastery.

Daily targets, local best evolution tracking, and a visible rank ladder make each restart feel like progress instead of a wipe.

Best Habit

Protect one anchor corner.

Late boards punish drifting. Build around one edge, keep your largest creature parked, and only break formation when a merge line is guaranteed.
