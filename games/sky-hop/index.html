How to play

Sky Hop is a climber. Your character bounces automatically when it lands on a cloud pad, so the goal is to guide each landing and keep the camera moving upward.

Controls

  • Keyboard: Left and Right arrows or A and D
  • Mouse: move across the canvas to guide the hop line
  • Touch: drag left and right on the canvas
  • Restart: press Space or Enter after a run ends
Game loop

Score comes from height and clean landings. Some cloud pads drift, so the run stays readable but never fully static.

Sky Hop tip

Watch the next pad early. The best runs come from lining up the next bounce before the current jump peaks.
